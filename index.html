My Projects

Apertr

A full stack project built using Flickr's site UI and functionality of several core features as the basis for wireframes and design of schema, state shape, and routes. Backend consists of Ruby on Rails and PostgreSQL while frontend uses React.js with the Redux workflow. Users can create a secure login, upload and store photos in AWS S3 buckets as well as create albums with their photos, create comments and tags for all photos, and search all photos based on a tag word.

Tap Tap Revolution

A web game app inspired by the classic Dance Dance Revolution, played using the direction keys. Built using HTML5 Canvas and vanilla JavaScript. Players have several options to customize their playing experience: mute/unmute, pause/unpause, and restart. The game will progressively pick up in difficulty and the higher your combo/streak, the higher your score increases per successful tap.

OverMatch

A full stack project built on the popular MERN (MongoDB, Express, React, Node) stack and features OAuth 2.0 to delegate authentication to Battle.net. Users login securely using their existing credentials, and algorithms extract information during login to use in other API calls to collect data to help create their dream team with fellow players.
